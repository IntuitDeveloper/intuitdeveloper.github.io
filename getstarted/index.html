Getting started with QuickBooks API

GET STARTED

  1. Create a developer account

  2. Create an app

  3. Generate OAuth tokens using playround

  4. Make the API call - click to see the curl request

    curl -X GET 'https://sandbox-quickbooks.api.intuit.com/v3/company/company id/companyinfo/company id/?minorversion=12' -H 'accept: application/json' -H 'authorization: Bearer accesstoken' -H 'content-type: application/json'
DEVELOPER TOOLS

  1. Explore API using API explorer

  2. Download samples

  3. Use SDK to code faster - DotNet - Java - PHP - Other

  4. Download postman collection

  5. Use sandbox for testing
